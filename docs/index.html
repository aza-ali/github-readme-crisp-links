How to use it in your README

  1. 1
    Customize

    Pick a color or gradient above. Live preview updates as you type.

  2. 2
    Save the SVG into your repo

    Download the SVG and drop it somewhere like assets/ or text/. Or generate it with the CLI: pip install crisp-links.

  3. 3
    Paste the markdown

    Copy the <a><img/></a> snippet into your README. GitHub will render the SVG without underlining it.

Batch mode

Same JSON format as the CLI's --batch flag. Generate all your project links in one go.

Run the batch to see SVGs and markdown snippets.
